## niri
I only included the parts of the config I deemed relevant, you'll have to merge them yourself
(and possibly adjust some values like monitors etc).

### Software
The bar is built with Quickshell.
As launcher I use rofi (config.rasi) and my terminal is Alacritty (alacritty.toml).
The wallpapers I use are swaybg and awww.
File manager is Nautilus.

### Fonts
- JetBrains Mono
- Nerd Fonts
- Montserrat
- Fredoka
